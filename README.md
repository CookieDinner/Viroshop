# PPI - Viroshop

Dev version of API: https://dev-ppi.herokuapp.com<br>
Prod version of API: https://viroshop.herokuapp.com (unavailable yet)<br>
<br>
To deploy backend application to heroku server:<br>
	1. Run herokuDeployDevPPI.bat<br>
	2. When you get message "RUN SCRIPT: herokuAdditionalDevPPI.sh" in console 	then run "herokuAdditionalDevPPI.sh"<br>
	3. In main console where you started bat file click something to continue 	process of deployment.<br>
	4. Done<br>
